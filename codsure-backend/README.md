# CODSure Backend

FastAPI-based backend for the CODSure platform.

## Features
-   **Risk Engine**: Analyzes order data to determine COD eligibility.
-   **Seller Management**: Auth, profile, and settings.
-   **API**: RESTful endpoints for frontend and integrations.

## Tech Stack
-   Python 3.11+
-   FastAPI
-   PostgreSQL
-   SQLAlchemy (Async)
-   Pydantic
